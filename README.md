![snapshot](resources/snapshot.png)

## Features

- 提供全部板块选择
- 小型的帖子列表
- Github 风格的帖子内容展示

## Extension Settings

This extension contributes the following settings:

- `neko.cookie`: 登录后 NGA 的 Cookie，需要在插件中设置。
